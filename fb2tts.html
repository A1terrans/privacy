Privacy Policy

Alterran built the FB2 TTS app as a Free app. This app is provided by Alterran at no cost and is intended for use as is.

Information Collection and Use

The application uses several extended permissions to access the file system. This is necessary for the correct functioning of the methods used in it for reading book files on your device. The application will only interact with directories and files with the fb2 extension on your device, everything else is ignored by the application. The application stores information about the history of reading books, the current progress of reading each book, illustrations of all read books and the path in the file system to the files of books. All the information contained in the application remains in the application. The application does not send any data.
